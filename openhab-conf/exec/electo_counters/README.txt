В electro_counter.conf указать нужные: DEVICE, COUNTER_SN, PASS_READ, COUNTER_TYPE, ECHOCOMMAND
запускать: sh electro_counter.sh <КОМАНДА> <НОМЕР СЧЕТЧИКА> <КОНФИГУРАЦИОННЫЙ ФАЙЛ>

<КОМАНДА> - если не указана - команда kwatthour:all_time:T_ALL
<НОМЕР СЧЕТЧИКА> - если не указан, берется из конфигурационного файла
<КОНФИГУРАЦИОННЫЙ ФАЙЛ> - если не указан - берется файл конфигурации по умолчанию electro_counter.conf


Перечень команд

Mercury 230

test              Тестирование связи
openread          Открытие соединения на чтение
openwrite         Открытие соединения на запись
close             Завершение сеанса
kwatthour         Опрос накопленной энергии (по умолчанию, если никакая команда явно не указана)
 kwatthour:MODE:ZONE
  MODE может быть:
   all_time - по умолчанию за все время
   curr_year - за текущий год
   prior_year - за предыдущий год
   month - за месяц
   today - за сегодня
   yesterday - за вчера
 ZONE - зоны\тарифы
   T_ALL - общая сумма по всем тарифам - по умолчанию
   T1
   T2
   T3
   T4
EXAMPLES
  kwatthour:today:T2 - накопленная энергия за текущие сутки по 2-му тарифу
  kwatthour:month=04:all_time - накопленная энергия за апрель по всем тарифам в сумме

amper             Сила тока A (А) по фазам
coefw             Коэффициент мощности (С) по фазам
freq              Частота Гц
coin              Угол между фазами
power             Мощность P (Вт) по фазам
volt              Напряжение U (В)
null              Произвольная команда (возможно описать любую команду)
serialnum         Серийный номер счетчика и даты выпуска
version           Версия ПО
datetime          Дата время по счетчику
kwatthour_phaza   Накопленная энергия по указанной фазе
power_phaza       Мощность P (Вт) по указанной фазе
